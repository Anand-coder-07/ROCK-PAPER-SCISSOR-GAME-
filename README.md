# 🪨📄✂️ Rock Paper Scissors Game

A simple and interactive **Rock Paper Scissors** game built using **HTML, CSS, and JavaScript**.

## 🎮 How to Play

1. Click on **Rock**, **Paper**, or **Scissors** to make your move.
2. The computer will randomly pick its move.
3. The result (Win / Lose / Tie) will be shown instantly.
4. Your score is **saved automatically** — even if you close the browser!
5. Click **Reset Score** to start fresh.

## 🚀 Features

- 🎲 Random computer move generation
- 💾 Score saved using `localStorage` (persists across sessions)
- 🎨 Glassmorphism-style UI with blur effects
- ✨ Smooth hover animations on buttons
- 🔴 Reset button to clear all scores

## 🛠️ Tech Stack

| Technology | Usage |
|------------|-------|
| HTML | Game structure & layout |
| CSS | Styling, animations, glassmorphism |
| JavaScript | Game logic & localStorage |

## 🖥️ How to Run

1. Clone or download this repository
2. Make sure `img.jpg` is in the same folder
3. Open `06-Rock-Paper-Scissors.html` in any browser
4. Start playing! 🎉
